 Управление выводом текста
 с использованием FontMetrics

 Java требует от программистов ручного управления практически всем выводом текста.

 С учетом того, что размер каждого шрифта может отличаться, а шрифты
 могут изменяться во время выполнения программы, должен существовать
 какой-нибудь способ выяснения размеров и других атрибутов шрифта, выбранного в текущий
 момент.
 Скажем, вывод одной строки текста после другой означает необходимость в наличии
 способа узнать высоту шрифта и количество пикселей между строками.
 Для удовлетворения такой потребности в библиотеке AWT предусмотрен класс FontMetrics,
 который инкапсулирует разнообразную информацию о шрифте.

 Высота             Размер строки текста сверху донизу
 Базовая линия      Линия, по которой выравниваются нижние
                    части символов (не считая спуска)
 Подъем             Расстояние между базовой линией и верхом символа
 Спуск              Расстояние между базовой линией и низом символа
 Интерлиньяж         Расстояние между низом одной строки текста
                     и верхом следующей
 _________________________________________________________________________
 В классе FontMetrics определены методы, помогающие управлять выводом текста.

int bytesWidth (byte[] b, int start , int numBytes)
/Возвращает ширину numвytes символов из массива Ь, начиная с элемента start/

int charsWidth(char[] с, int start, int numChars )
/Возвращает ширину numChars символов из массива с, начиная с элемента start/

int charWidth(char с)      /Возвращает ширину символа с/

int charWidth(int с)       /Возвращает ширину символа с/

int getAscent()            /Возвращает подъем шрифта/

int getDescent()           /Возвращает спуск шрифта/

Font getFont()             /Возвращает шрифт/

int getHeight()            /Возвращает высоту строки текста. Это значение
                            можно использовать для многострочного вывода текста в окне/

int getLeading()          /Возвращает расстояние между строками текста (т.е. интерлиньяж)/

int getMaxAdvance()        /Возвращает оценочную ширину самого широкого символа или -1,
                             если это значение недоступно /
int getMaxAscent()         /Возвращает максимальный подъем шрифта /

int getMaxDescent()        /Возвращает максимальный спуск шрифта/

int [] getWidths()         /Возвращает ширину первых 256 символов/

int stringWidth( String str)   /Возвращает ширину строки, указанной в str/

String toString()          /Возвращает строковый эквивалент вызывающего объекта/
____________________________________________
класс FontMetrics чаще всего применяется для определения
расстояния между строками текста. Второй распространенный
сценарий использования касается выяснения длины отображаемой строки.

В общем случае для отображения нескольких строк текста в программе
придется вручную отслеживать текущую позицию вывода. Каждый раз, когда
требуется новая строка, координата У должна быть перемещена в начало
следующей строки. Каждый раз, когда отображается строка, координата Х
должна быть установлена в точку, где строка заканчивается. В итоге
следующую строку можно вывести так, чтобы она начиналась в конце предыдущей
строки.

Часто нужно знать лишь общую высоту строки, которая представляет собой сумму
значений интерлиньяжа, подъема и спуска шрифта.
