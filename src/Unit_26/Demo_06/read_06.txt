Создание и выбор шрифта

Для создания нового шрифта понадобится сконструировать описывающий
его объект Font. Конструктор класса Font имеет следующую общую форму:

Font(String fontName, int fontStyle, int pointSize)
- fontName      /имя желаемого шрифта/
                Все среды Java будут поддерживать шрифты Dialog (по умолчанию),
                DialogInput, SansSerif, Serif и Monospaced. Можно задействовать
                любые другие шрифты, поддерживаемые в конкретной среде, но
                следует проявлять осторожность, т.к. они могут быть недоступными повсеместно.
- fontStyle     /начертание шрифта/
                Оно может включать одну или несколько констант Font.PLAIN,
                Font.BOLD и Font.ITALIC,
                для комбинирования которых применяется операция ИЛИ. Скажем,
                Font.ВOLD | Font.ITALIC задает полужирное курсивное начертание.
- pointSize     /размер шрифта в пунктах/
________________________________________________________________________________
Чтобы использовать созданный шрифт, его нужно выбрать с помощью метода setFont(),
который определен в Component и имеет такой общий вид:

void setFont (Font fontObj )    //В fontObj передается объект, содержащий желаемый шрифт.
