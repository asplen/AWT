Использование вставок
--------------------------------------------------------------
Иногда может требоваться оставлять небольшое пространство между
контейнером, содержащим компоненты, и окном, в котором они находятся,
для чего необходимо переопределить метод getInsets () класса Container.
Данный метод возвращает объект типа Insets, содержащий верхнюю,
нижнюю, левую и правую вставки, которые будут применяться при
отображении контейнера. Такие значения используются диспетчером компоновки для
вставки компонентов при компоновке окна.


Insets( int top, int left, int bottom, int right)

Insets getInsets ()

В случае переопределения метода getInsets () понадобится возвратить
новый объект Insets, который содержит желаемый промежуток для вставки.
