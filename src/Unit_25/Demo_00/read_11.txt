Класс ItemEvent
_________________________________________________________________________
Событие типа ItemEvent генерируется при щелчке на флажке или
элементе списка, либо при выборе или отмене выбора
отмечаемого пункта меню.

Существуют два типа событий элементов, которые
идентифицируются целочисленными константами:

1   DESELECTED      Пользователь отменил выбор элемента
2   SELECTED        Пользователь выбрал элемент

Кроме того, в классе ItemEvent определена целочисленная константа
ITEM_STATE_CНANGED, которая обозначает изменение состояния.

ItemEvent(ItemSelectable src, int type, OЬject entry, int state)

src     передается ссылка на компонент, сгенерировавший событие.
type    Тип события
entry   элемент, сгенерировавший событие элемента,
state   Текущее состояние этого элемента
_________________________________________________________________________

Object getItem ()                   получить ссылку на измененный элемент.

ItemSelectable getItemSelectable()  получения ссылки на объект ItemSelectable, сгенерировавший событие
                                    Списки и переключатели являются примерами элементов пользовательского
                                    интерфейса, которые реализуют интерфейс ItemSelectable.

int getStateChange()                возвращает изменение состояния
                                    SELECTED или DESELECTED
